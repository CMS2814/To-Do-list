CLI To-Do List app

A command-line To-Do List app built in python that lets users Add, View, Edit, and Delete task. All data is stored locally using a JSON file.

Features:
-Add new tasks with name, description, due date, priority, and status
-View all tasks or a specific task by ID
-Edit existing tasks (name, description, due date, priority, or status)
-Delete tasks by ID
-Data saving using a local JSON file
-Input validation and basic error handling

How to run:
1. Clone this repository or download the files
2. Run Main.py
3. Follow the on-screen instructions.
